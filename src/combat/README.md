# Combat

Combat engine and encounter simulation logic.

Examples of modules to implement here:
- **Initiative** - Turn order determination and tracking
- **Actions** - Attack, cast spell, dodge, disengage, dash, and other combat actions
- **Damage** - Damage calculation, resistances, vulnerabilities, and immunities
- **Encounter** - Encounter setup, round management, and win/loss conditions
- **AI** - Decision-making logic for NPC/monster combat behavior
