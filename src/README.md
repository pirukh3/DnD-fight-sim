# Source Code

Core application source code for the DnD Fight Simulator.

## Subdirectories

- **models/** - Data models for characters, monsters, spells, items, and other game entities
- **combat/** - Combat engine logic including initiative, attack rolls, damage calculation, and status effects
- **utils/** - Shared utility functions such as dice rolling, logging, and configuration helpers
- **ui/** - User interface layer (CLI or GUI) for interacting with the simulator
